bitSkin
=======

to run this algorithm you need this software: rhinoceros > grasshoper > firefly and python 2.7
this hardware: arduino uno, rotary encoder and a push button.

Pinout:
pin_1: Arduino pin 3,
pin_2: Arduino pin 2,
pin_3: Arduino pin GND,
pin_4: Arduino pin 7
